# Supply Chain Agent with Memory (Vertex AI Memory Bank)

In this codelab, you will build a Secure Supply Chain Orchestrator. Building on the foundations of multi-agent orchestration and high-performance data retrieval, you will introduce Google Cloud Model Armor—a specialized security layer designed to protect LLM-powered applications. You will learn how to "sandwich" your agentic logic between two security layers: an Input Shield to block malicious prompt injections and an Output Shield to prevent the accidental leakage of sensitive supply chain data or PII (Personally Identifiable Information).
